# 💱 Currency Converter

A modern currency converter built with HTML, CSS, and JavaScript.

## Features

- 🌍 Live exchange rates
- 💵 Convert between multiple currencies
- 🎨 Modern glassmorphism interface
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🔗 Uses a public exchange rate API

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API
